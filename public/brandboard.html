12 — Guidelines

Brand Usage Rules

✓ Do

  • Use Khand Bold for all headings, always uppercase
  • Use Alex Brush for elegant accent phrases only
  • Keep body text in Prompt at 300–600 weight
  • Maintain wide letter-spacing on labels (0.15em–0.3em)
  • Use the gold primary (#BDA887) sparingly for accents
  • Pair dark backgrounds with light text
  • Use border-top 4px primary on cards
  • Maintain generous whitespace between sections

✗ Don't

  • Don't use Alex Brush for body copy or long text
  • Don't apply colors outside the defined palette
  • Don't use lowercase for Khand display headings
  • Don't reduce letter-spacing on navigation links
  • Don't use rounded corners on primary buttons
  • Don't pair gold text on light backgrounds without contrast
  • Don't use more than 2 font weights on a single card
  • Don't skip the label pattern above section headings
